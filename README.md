# AuditFlow – Frontend

AuditFlow Frontend is a modern, responsive web interface designed to visualize and manage audit logs efficiently. It focuses purely on user experience, performance, and clean UI.

## Tech Stack
- React
- Vite
- Tailwind CSS
- ESLint

## Features
- Clean and minimal user interface
- Responsive design for all screen sizes
- Component-based architecture
- Fast development with Vite
- Utility-first styling using Tailwind CSS
